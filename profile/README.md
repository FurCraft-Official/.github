# 🛠️ FurCraft-Official

欢迎访问 **FurCraft-Official** 的官方 GitHub 主页！我们是一个开源开发团队，致力于构建有趣、有用、并富有创意的项目，包括 Minecraft 相关工具与 API 服务、实用脚本等。👋

🔗 社区网站: [https://furcraft.top](https://furcraft.top)

📧 联系邮箱: [admin@furcraft.top](mailto:admin@furcraft.top)

## 🚀 关于我们

**FurCraft-Official** 是一个开放参与的开发者组织，目标是通过协作与创意推动有趣的项目发展。无论是API 接口、开发工具还是学习资源，我们都欢迎有兴趣的开发者与我们一起共创优秀项目。

---

## 📦 我们的项目

下面是我们目前部分公开的仓库：

### ⭐ 核心项目

* **node‑imgapi** — Node.js 版图片 API_V1

* **Fur‑Img‑API_V2** — 一个轻量级的随机图片 API 服务

* **Fur-img-api-next** — 人工重写的随机图片API服务

* **FurAPI‑mc** — 查询 Minecraft 游戏或服务器状态及信息

* **GMI** — 精神健康相关指南项目（Forked）

> ⚠️ 项目列表会随着更新变化，请查看 Repositories 页面了解最新内容。
